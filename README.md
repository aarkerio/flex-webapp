# flex-webapp

## Prerequisites

You will need [Leiningen][1] 2.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Install

    git clone git@github.com:aarkerio/flex-webapp.git && cd flex-webapp

## Running

Start the web server:

    lein run

In another console tab:

    lein figwheel

Test:

    lein test
